# Сайт проекта Perfect Coffee Project
Цель проекта: создание удобного сервиса для оценки кофе.
На данном этапе представлена главная страница будущего сервиса,
которая позволяет оценить дизайн, адаптивность и расположение элементов.

## Что внутри
Шапка с навигацией;
Ссылки на блоки страниц, посвященных конкретному кофе;
Кнопка добавления отзыва;
Макет таблицы лидеров проекта;
Макет фильтрации таблицы лидеров;
Блок доверия с небольшим отсуплением;
Номер телефона для связи

## Технологии
Использование HTML5
CSS: использование Tailwindcss
